This is a simple CRUD REST API for a bookstore   (Django/Python)



Implemented with class based views


METHOD       GET
ROUTE        /api/books
METHOD       GET
ROUTE        /api/books/{id}
METHOD       POST
ROUTE        /api/books
EXAMPLE       {"title":"title1","author":"author1","price":1.0}
METHOD       PUT
ROUTE        /api/books/{id}
METHOD       DELETE
ROUTE        /api/books/{id}


Implemented with function based views


METHOD       GET
ROUTE        /funcbased/api/books
METHOD       GET
ROUTE        /funcbased/api/books/{id}
METHOD       POST
ROUTE        /funcbased/api/books
EXAMPLE       {"title":"title1","author":"author1","price":1.0}
METHOD       PUT
ROUTE        /funcbased/api/books/{id}
METHOD       DELETE
ROUTE        /funcbased/api/books/{id}

  • When using POST method, do not specify the "id" property because it is autoincremental.
